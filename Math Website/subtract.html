How to Subtract Numbers!

To subtract numbers, you have to know how to count your numbers from at least 1-20! Start at the first number and go backward the number of time of the second number. You can use a number line to help you if you want.

Ex.: 7-5

For the example above, we would start at 7 and go back 5 times. So we would start at 7...then 6...5...4...3...2
So 2 is our answer! As you practice more and more, you will eventually memorize subtracting these single digit numbers together.
