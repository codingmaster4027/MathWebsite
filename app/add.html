How to Add Numbers!

To add numbers, you have to know how to count your numbers from at least 1-20! Start at the first number and count forward the number of time of the second number. You can use a number line to help you if you want.

For example, in the image above, we have the problem 9+7. We start at 9 and go forward 7 times (represented by the pink). We land on 16. Therefore, 9+7 is 16.

Ex.: 2+5

For the example above, we would start at 2 and count up 1 5 times. So we would start at 2...then 3...4...5...6...7.
So 7 is our answer! As you practice more and more, you will eventually memorize adding these single digit numbers together.
